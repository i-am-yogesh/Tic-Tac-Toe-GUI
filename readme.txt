HOW TO PLAY:
** This is mutiplayer game. No CPU mode is provided.

1. (Optional) Enter names of players.
2. Press start button. Game will not start until start button is clicked.
3. Click cell you want to mark.



RULES:

1. The game is played on a grid that's 3x3 squares.

2. 'X' is always given first turn, '0' has second turn. Players take turns putting their marks in empty squares.

3. The first player to get 3 of his/her marks in a row (up, down, across, or diagonally) is the winner.

4. When all 9 squares are full, the game is over. If no player has 3 marks in a row, the game ends in a tie.
